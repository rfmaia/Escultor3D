Detailed Description

A estrutura de dado Voxel serve para armazenar a cor e transparência de um voxel.

As variáveis que armazenam a cor rgba podem assumir valores entre 0 e 1, enquanto isOn recebe um valor booleano TRUE ou FALSE para indicar se o voxel está ativo ou não.
